### 2. ⚠️ Изображения требуют замены

Текущие изображения (`slide-1.png` ... `slide-7.png`) — это временные цветные заглушки. 

**Для получения правильных изображений из Figma:**

1. **Откройте макет в Figma:**
   - https://www.figma.com/design/z9cmA4j5jwS5PUgrSuY0z5/Tracks?node-id=57-41347
   - Найдите секцию 2 "Управляйте своим психологическим состоянием"
   - Найдите карусель с 7 карточками

2. **Для каждой карточки:**
   - Выберите карточку (весь блок с цветным фоном и мокапом телефона)
   - Правый клик → **Export** → **PNG** → **2x** → **Export**
   - Сохраните в `public/images/` с правильным именем:
     - `slide-1.png` - Тесты
     - `slide-2.png` - Трекер состояния
     - `slide-3.png` - Статьи
     - `slide-4.png` - Аудиотренировки
     - `slide-5.png` - Восстановительные ритмы
     - `slide-6.png` - Медитации
     - `slide-7.png` - Практики самопомощи

3. **Или используйте скрипт автоматического экспорта:**
   ```bash
   export FIGMA_ACCESS_TOKEN="your-token"
   python3 export_carousel_slides.py
   ```

## Как проверить

1. **Обновите страницу с очисткой кэша:**
   - Mac: `Cmd + Shift + R`
   - Windows/Linux: `Ctrl + Shift + R`

2. **Проверьте цвета карточек:**
   - Каждая карточка должна иметь свой уникальный пастельный цвет фона
   - Карточки больше не все жёлтые

3. **После замены изображений:**
   - Каждая карточка должна показывать правильный экран приложения из макета
   - Изображения должны соответствовать содержимому карточки
