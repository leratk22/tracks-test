# Ручной экспорт изображений слайдов из Figma

Если автоматический экспорт не работает, выполните следующие шаги:

## Способ 1: Экспорт через Figma Desktop (рекомендуется)

1. **Откройте файл в Figma Desktop:**
   - https://www.figma.com/design/z9cmA4j5jwS5PUgrSuY0z5/Tracks
   - Или откройте через приложение Figma Desktop

2. **Найдите секцию 2 "Управляйте своим психологическим состоянием"**
   - Найдите карусель с 7 карточками

3. **Для каждого слайда:**
   - Выберите карточку слайда (весь блок с жёлтым фоном и мокапом телефона)
   - Правый клик → **Export** → выберите **PNG** → **2x** → **Export**
   - Сохраните файл в папку `public/images/` с соответствующим именем:
     - **slide-1.png** - Тесты (первая карточка)
     - **slide-2.png** - Трекер состояния (вторая карточка)
     - **slide-3.png** - Статьи (третья карточка)
     - **slide-4.png** - Аудиотренировки (четвёртая карточка)
     - **slide-5.png** - Восстановительные ритмы (пятая карточка)
     - **slide-6.png** - Медитации (шестая карточка)
     - **slide-7.png** - Практики самопомощи (седьмая карточка)

## Способ 2: Экспорт через веб-интерфейс Figma

1. Откройте файл в браузере: https://www.figma.com/design/z9cmA4j5jwS5PUgrSuY0z5/Tracks?node-id=57-41347

2. Найдите секцию 2 с каруселью

3. Для каждого слайда:
   - Выберите карточку
   - В правой панели найдите раздел **Export**
   - Нажмите **+** → выберите **PNG** → **2x** → **Export**
   - Сохраните с правильным именем в `public/images/`

## Способ 3: Автоматический экспорт через API

Если у вас есть Figma Access Token:

```bash
# Установите токен
export FIGMA_ACCESS_TOKEN="your-token-here"

# Запустите скрипт
python3 export_carousel_slides.py
```

Получить токен: https://www.figma.com/developers/api#access-tokens

## Проверка

После экспорта проверьте:
1. Все 7 файлов `slide-1.png` ... `slide-7.png` находятся в `public/images/`
2. Каждое изображение уникально (разные экраны приложения)
3. Размер изображений примерно 606×1066 пикселей (для @2x)
4. Обновите страницу в браузере (Cmd+R) для проверки

## Важно

- Каждый слайд должен показывать **разный** экран приложения
- Изображения должны быть в формате PNG
- Рекомендуемый размер: 2x (606×1066px) для чёткости на retina-дисплеях
