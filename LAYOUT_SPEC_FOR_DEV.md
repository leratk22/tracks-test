# Спецификация вёрстки: лендинг Tracks (Figma 57:41347)

Документ для агента-верстальщика. Описание по секциям: структура блоков, скролл, компоновка.

**Figma:** [Tracks](https://www.figma.com/design/z9cmA4j5jwS5PUgrSuY0z5/Tracks?node-id=57-41347) — node-id `57-41347` (корневой фрейм страницы).

---

## Общая страница

- **Вертикальный скролл:** вся страница — одна длинная лента; скролл только по вертикали (одна колонка секций друг под другом).
- **Ширина контента:** по макету — контент может быть full-width в секциях или с max-width + центрированием; уточнять по Figma (направляющие, отступы).
- **Секции:** три основные секции, идут последовательно сверху вниз. Между секциями разрыв или нет — смотреть по макету.

---

## Секция 1 — «Моментальная поддержка»

- **Скролл:** только общий вертикальный скролл страницы. Внутри секции горизонтального скролла нет.
- **Фон:** светлый (кремовый/бежевый), возможны декоративные элементы (абстрактные фигуры розового/оранжевого) — фоновый слой или псевдоэлементы/отдельный блок.
- **Структура блока:**
  - Верх: заголовок «Моментальная поддержка» (крупный, жирный).
  - Под заголовком: абзац текста (описание преимуществ приложения).
  - Центр: мокап смартфона с изображением (женщина с телефоном), по макету — центральный визуальный якорь.
  - Вокруг мокапа: набор карточек/бейджей (круглые или прямоугольные) с подписями: «Делегирование», «Психология», «Вебинар», «Коуч», «Коктейль» и т.д., плюс иконка (например, солнце). Расположение — по макету (сверху слева/справа, по бокам, снизу).
- **Вёрстка:** блок можно собирать как flex/grid: колонка (заголовок + текст), затем контейнер с мокапом по центру и карточки абсолютно/относительно позиционированы вокруг или в одной сетке. Точные отступы и размеры — из Figma.
- **Адаптив:** на мобильных карточки вокруг мокапа могут уходить в колонку под текст или в упрощённый вариант; необходимость и брейкпоинты — по макету.

---

## Секция 2 — «Управляйте своим психологическим состоянием»

- **Скролл:**
  - По вертикали — секция участвует в общем скролле страницы.
  - По горизонтали — внутри секции горизонтальная карусель (скролл или слайдер): ряд карточек с мокапами экранов приложения, листание влево/вправо.
- **Фон:** белый.
- **Структура блока:**
  - Слева: заголовок «Управляйте своим психологическим состоянием».
  - Справа от заголовка (или под на мобильных): абзац с описанием (тот же смысл, что в первой секции).
  - Ниже: зона карусели.
    - Элементы карусели — карточки. В каждой: мокап телефона + заголовок карточки + краткое описание.
    - Заголовки карточек по макету: «Тесты», «Трекер состояния», «Статьи», «Аудиотренировки и релаксации», «Восстановительные ритмы», «Медитации», «Практики самопомощи».
  - Навигация карусели: две кнопки — «влево» и «вправо» (стрелки).
- **Вёрстка:** заголовок и текст — одна или две колонки (flex/grid). Карусель — горизонтальный скролл-контейнер (`overflow-x: auto` или слайдер на JS) с рядом карточек одинаковой ширины; кнопки стрелок скроллят или переключают слайды. Высота блока задаётся контентом карусели.
- **Адаптив:** на узких экранах заголовок и описание лучше в одну колонку; карусель остаётся горизонтальной, при необходимости скрыть стрелки и оставить только touch-скролл — по макету.

---

## Секция 3 — «Пошаговые планы самоподдержки»

- **Скролл:** только общий вертикальный скролл страницы. Сетка карточек в одну или несколько строк без горизонтального скролла (все карточки в потоке).
- **Фон:** верх секции — жёлтый баннер (полоса с заголовком и текстом); под баннером — белый фон.
- **Структура блока:**
  - **Баннер (жёлтый):**
    - Заголовок: «Пошаговые планы самоподдержки».
    - Под ним: текст «Проверенные инструменты, подобранные профессионалами…» (полный текст взять из Figma).
  - **Сетка карточек:** под баннером, сетка 3 колонки × 3 ряда (9 карточек).
    - Одна карточка: область изображения (плейсхолдер или картинка), заголовок (#Title / название), описание (#Description), цена «500 ₽», две кнопки — «Подробнее» и «Купить».
    - Карточки одинаковые по структуре и стилю.
- **Вёрстка:** баннер — на всю ширину секции. Сетка — CSS Grid (например, `grid-template-columns: repeat(3, 1fr)`) или flex с переносом; отступы между карточками — из Figma. Внутри карточки: блок изображения, заголовок, описание, блок с ценой, ряд из двух кнопок.
- **Адаптив:** на планшетах/мобильных — 2 колонки или 1 колонка; брейкпоинты и порядок — по макету.

## Дополнительно для вёрстки

- **Типографика и отступы:** все размеры шрифтов, межбуквенные/межстрочные, отступы между секциями и внутри блоков — брать из Figma (стили текста, авто-лейаут).
- **Цвета и скругления:** палитра (кремовый, жёлтый, оранжевый, розовый, чёрный текст, белый фон) и радиусы кнопок/карточек — из стилей и переменных Figma.
- **Интерактивность:** карусель (клик по стрелкам, при необходимости свайп); кнопки «Подробнее», «Купить», «Попробовать» — без изменений логики, только разметка и стили по макету.
