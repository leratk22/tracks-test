# Инструкция по замене изображений слайдов карусели

## Текущее состояние

В папке `public/images/` находятся временные изображения слайдов (`slide-1.png` - `slide-7.png`), которые являются копиями одного файла.

## Как получить правильные изображения из Figma

### Вариант 1: Через Figma Desktop (рекомендуется)

1. **Запустите Figma Desktop с remote debugging:**
   ```bash
   open -a "Figma" --args --remote-debugging-port=9222
   ```

2. **Откройте файл Tracks в Figma Desktop**

3. **Экспортируйте каждый слайд:**
   - Перейдите к узлам слайдов в макете (секция 2, карусель)
   - Node IDs слайдов:
     - `21:32131` - Тесты → сохранить как `slide-1.png`
     - `22:33546` - Трекер состояния → `slide-2.png`
     - `21:32583` - Статьи → `slide-3.png`
     - `22:36870` - Аудиотренировки → `slide-4.png`
     - `22:35178` - Восстановительные ритмы → `slide-5.png`
     - `22:36109` - Медитации → `slide-6.png`
     - `22:35183` - Практики самопомощи → `slide-7.png`

4. **Для каждого слайда:**
   - Выберите узел в Figma
   - Правый клик → Export → PNG → 2x
   - Сохраните в `public/images/` с соответствующим именем

### Вариант 2: Через Figma REST API

Если у вас есть `FIGMA_ACCESS_TOKEN`:

```bash
export FIGMA_ACCESS_TOKEN="your-token-here"
```

Затем используйте скрипт для автоматической загрузки изображений.

### Вариант 3: Ручной экспорт через веб-интерфейс

1. Откройте файл в браузере: https://www.figma.com/design/z9cmA4j5jwS5PUgrSuY0z5/Tracks
2. Для каждого слайда в секции 2:
   - Выберите карточку слайда
   - Нажмите Export в правой панели
   - Выберите PNG, 2x
   - Скачайте и переименуйте согласно списку выше

## Проверка

После замены изображений:
1. Обновите страницу в браузере (Cmd+R)
2. Проверьте, что каждый слайд карусели показывает уникальное изображение
3. Убедитесь, что изображения загружаются без ошибок (откройте DevTools → Network)

## Размеры изображений

- Рекомендуемый размер: 303×533 пикселей (или 606×1066 для @2x)
- Формат: PNG с прозрачностью (если нужно) или JPG
- Оптимизация: сжимайте изображения для веба (TinyPNG, ImageOptim и т.п.)
